# 技术栈说明（非核心）

本文件仅说明软件分发和运行所需的非核心技术信息，不包含源码、核心实现细节或内部工程配置。

## 软件形态

- Windows 桌面端单文件 exe
- 本地运行，无需用户手动安装开发环境
- 通过 GitHub Release 分发安装包

## 界面与交互

- 桌面图形界面
- 支持 SMILES 文本输入和结构绘制入口
- 支持 PI/PBZ 体系选择
- 支持 Dieletric Constant、Dieletric Loss、Td5%、Tg、SA score 等性能结果展示
- 支持说明弹窗、主题切换和中英文切换

## 发布与更新

- 当前公开 Release 保留 V 系列与 T 系列安装包
- V 系列识别：`ScientificPlatformPrediction_*.exe`
- T 系列识别：`ScientificPlatformPrediction_T*.exe`
