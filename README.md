# 研构智库发布说明

本仓库仅用于发布软件下载说明和安装包，不存放项目源码、核心算法、模型细节或数据内容。

## 下载地址

请进入 GitHub Release 下载最新版：

- [研构智库 V1.3.2 / T1.3](https://github.com/dilucyad/Prediction-repository/releases/tag/V1.3.2)

## 当前版本

- V 系列稳定版：`ScientificPlatformPrediction_V1.3.2.exe`
- T 系列预测版：`ScientificPlatformT_T1.3.exe`

Release 页面仅保留最新一组发布文件：V 系列一份、T 系列一份。

## V 系列 V1.3.2

V 系列为稳定分发版本，面向日常使用场景。主要功能包括：

- SMILES 输入与结构绘制
- 结果展示与预测提示
- 中英文切换
- 主题切换
- 软件说明弹窗
- 版本更新检测

## T 系列 T1.3

T 系列为预测功能版本，面向直接预测场景，暂时只预测 PI 的性能。主要功能包括：

- SMILES 输入与结构绘制
- 温度、频率等条件输入
- Dieletric Constant、Dieletric Loss、Td5%、Tg 结果展示
- SA score 计算与展示
- 中英文切换
- 主题切换
- 软件说明弹窗
- 版本更新检测

## 更新规则

V 系列和 T 系列分别识别各自的安装包：

- V 系列只识别 `ScientificPlatformPrediction_*.exe`
- T 系列只识别 `ScientificPlatformT_*.exe`

两个系列可以放在同一个 Release 中，但更新时只会提示和下载对应系列的新版本。

## 使用方式

1. 打开 Release 页面。
2. 根据需要下载 V 系列或 T 系列 exe 文件。
3. 双击运行程序。
4. 在界面中输入 SMILES 或绘制结构，并查看结果。
